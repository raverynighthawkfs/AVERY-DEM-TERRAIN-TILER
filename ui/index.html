Choose an input DEM (e.g. GeoTIFF) and an output directory. Tile size and max level can be adjusted as needed.

Quick start & supported files

  1. Select a GeoTIFF DEM file (.tif, .tiff) as the input.
  2. Choose an output directory (an empty folder is recommended).
  3. Adjust tile size and Z_LEVEL (max level of detail) if needed, then click Generate Tiles.

Currently, only GeoTIFF-based DEMs are supported. Other DEM formats should be converted to GeoTIFF before loading. This build is a prototype: the generator logs configuration and does not yet write real terrain tiles.
